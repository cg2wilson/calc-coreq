    Solve the following equations:
    1. \(\displaystyle \dfrac{1}{x+1}=\dfrac{3}{10}\)
    2. \(\displaystyle \dfrac{8}{2x} = x-3\)
    3. \(\displaystyle \dfrac{1}{z-5} = 0\)
    4. \(\displaystyle \dfrac{t-2}{t+3} = 0\)

    3. No solution
    4. \(\displaystyle t=2\)
    5. \(\displaystyle y = \dfrac{-1\pm \sqrt{5}}{2}\)
    Example 4.1.2.
    Solve the following equations
    1. \(\displaystyle \dfrac{2}{x} + 5 = 7\)
    2. \(\displaystyle \dfrac{t}{2} = \dfrac{4t}{3t-1}\)
    3. \(\displaystyle \dfrac{z^2+1}{z} = 0\)
    Solution.
    1. \(\displaystyle x=1\)
    2. \(\displaystyle t=0,3\)
    3. No solution
    Example 4.1.3.
    Solve the following equations
    1. \(\displaystyle \dfrac{2x+1}{x+10}=0\)
    2. \(\displaystyle \dfrac{2}{y+2} + \dfrac{1}{y-10} = 1\)
    Solution.
    1. \(\displaystyle x=-\dfrac{1}{2}\)
    2. \(\displaystyle y=-0.179,11.179\)

    Subsubsection 4.1.4 Solving Exponential Equations

    Example 4.1.9.
    Video for a‚Äâ12‚Äâ Video for b‚Äâ13‚Äâ Video for c‚Äâ14‚Äâ
    Solve the following equations
    1. \(\displaystyle e^{x/2} - 1 = 5\)
    2. \(\displaystyle 4\cdot 2^{5x} = 12\)
    3. \(\displaystyle e^x = 3^{x+1}\)
    Solution.
    1. \(\displaystyle x=2\ln 6\approx 3.584\)
    2. \(\displaystyle x=\dfrac{\log_2(3)}{5}\approx 0.317\)
    3. \(\displaystyle x=\dfrac{\ln 3}{1-\ln 3}\approx -11.141\)
    Example 4.1.10.
    Solve the following equations
    1. \(\displaystyle 5.7e^t = 11.2\)
    2. \(\displaystyle 3^{2x-7}+3=9\)
    3. \(\displaystyle e^x = 7^{5-x}\)
    Solution.
    1. \(\displaystyle t = \ln\lrpar{\dfrac{11.2}{5.7}}\approx 0.675\)
    2. \(\displaystyle x = \dfrac{\log_3(6)+7}{2}\approx 4.315\)
    3. \(\displaystyle x = \dfrac{5\ln 7}{1+\ln 7}\approx 3.303\)

    Solve the following equations
