    1. Factor \(y=x^2-2x\)
    2. Factor \(f(x) =3x^2+12x\)

    Factor \(y=x^2+10x\)
    Solution.

    Subsubsection 3.1.2 Factoring: Difference of Squares

    Example 3.1.3.
    Video‚Äâ2‚Äâ
    1. Difference of squares format
    2. Factor \(g(x) = x^2-4\)
    Solution.
    1. Difference of squares format
    2. \(\displaystyle x^2-4 = (x+2)(x-2)\)
    Example 3.1.4.
    Factor \(f(x) = x^2-49\)
    Solution.
    \(x^2-49 = (x-7)(x+7)\)

    Subsubsection 3.1.3 Factoring: Lead Coefficient \(=1\) (\(+/-\) subcases)

    Example 3.1.5.
    Video‚Äâ3‚Äâ
    1. Factor \(y=x^2+5x+6\)
    2. Factor \(y=x^2-7x+10\)
    3. Factor \(y=x^2-4x-12\)
    Solution.
    1. \(\displaystyle y=x^2+5x+6=(x+2)(x+3)\)
    2. \(\displaystyle y=x^2-7x+10=(x-2)(x-7)\)
    3. \(\displaystyle y=x^2-4x-12=(x-6)(x+2)\)
    Example 3.1.6.
    Factor all.
    1. \(\displaystyle y=x^2+8x+15\)
    2. \(\displaystyle y=x^2-x-2\)
    3. \(\displaystyle y=x^2+2x-8\)
    Solution.
    1. \(\displaystyle x^2+8x+15=(x+3)(x+5)\)
    2. \(\displaystyle x^2-x-2=(x-2)(x+1)\)
    3. \(\displaystyle x^2+2x-8=(x+4)(x-2)\)

    Subsubsection 3.1.4 Factoring: Lead Coefficient \(\neq 1\) ("magic X" method)

    Example 3.1.7.
    Video‚Äâ4‚Äâ
    1. Factor \(f(x) = 2x^2+13x+15\)
    2. Factor \(h(t) = 6t^2-5t+1\)
    Solution.
    1. \(\displaystyle 2x^2+13x+15 = (2x+3)(x+5)\)
    2. \(\displaystyle 6t^2-5t+1 = (2t-1)(3t-1)\)

    Factor \(g(x)=4x^2+19x-5\)
    Solution.
    \(4x^2+19x-5 = (4x-1)(x+5)\)

    Subsection 3.2 Solving Quadratic Equations

    Supplements XYZ Sections: 3.4 Derivatives as Rates of Change, 4.4 Mean Value Theorem, 4.4 First and Second Derivative Test

    Subsubsection 3.2.1 Solve by Factoring

    Example 3.2.1.
    Video‚Äâ1‚Äâ
    Find where \(f(x)=0\) given \(f(x) =x^2-3x+2\)
    Solution.
    \(x=1,x=2\)
    Example 3.2.2.
    Solve \(x^2+7x=-6\)
    Solution.
    Add 6 to both sides (one side must be zero):
    \begin{equation*} x^2+7x+6=0 \end{equation*}
    Factor:
    \begin{equation*} (x+6)(x+1)=0 \end{equation*}
    Set each factor equal to zero and solve:
    \begin{align*} x + 6 = 0 \amp\rightarrow x=-6 \\ x + 1 = 0 \amp\rightarrow x=-1 \end{align*}

    Subsubsection 3.2.2 Root Method

    Example 3.2.3.
    Video‚Äâ2‚Äâ
    1. Solve \(x^2-7=0\) for \(x\) exactly.
    2. Solve \(4(x-1)^2=18\) for \(x\) exactly.
    Solution.
    1. \(\displaystyle x=\pm \sqrt{7}\)
    2. \(\displaystyle x = 1\pm \dfrac{3}{\sqrt{2}}\)
    Example 3.2.4.
    Solve \(2x^2-6=0\) for \(x\) exactly.
    Solution.
    Isolate the squared term:
    \begin{equation*} 2x^2=6\rightarrow x^2=3 \end{equation*}
    Square root both sides:
    \begin{equation*} x=\pm \sqrt{3} \end{equation*}

    Subsubsection 3.2.3 Quadratic Formula

    Example 3.2.5.
    Video‚Äâ3‚Äâ
    For a quadratic equation of the form \(ax^2+bx+c=0\text{,}\) we can solve using the quadratic formula:
    \begin{equation*} x = \dfrac{-b\pm \sqrt{b^2-4ac}}{2a} \end{equation*}
    • 2 real solutions if \(b^2-4ac \gt 0\)
    • 1 real solution if \(b^2-4ac = 0\)
    • 0 real solutions if \(b^2-4ac \lt 0\)
    1. Solve \(2.1x^2+5.2x=1.3\) for \(x\text{.}\) Round to the nearest hundredth.
    2. Solve \(3t^2-7t-1=0\) for \(t\text{.}\) Round to the nearest hundredth.
    Solution.
    1. \(\displaystyle x=0.23, x=-2.71\)
    2. \(\displaystyle x=2.47, x=-0.14\)
    Example 3.2.6.
    1. Solve \(x^2-11x-2=0\) for \(x\text{.}\) Round to the nearest hundredth.
    2. Solve \(1.9x^2+7.2x=-2.1\) for \(x\text{.}\) Round to the nearest hundredth.
    Solution.
    1. \begin{equation*} x=\dfrac{-(-11)\pm \sqrt{(-11)^2-4(1)(-2)}}{2(1)}\approx -0.18,11.18 \end{equation*}
    2. \begin{equation*} x=\dfrac{-7.2\pm \sqrt{(7.2)^2-4(1.9)(2.1)}}{2(1.9)}\approx -3.47,-0.32 \end{equation*}
