    Subsubsection 5.1.1 Adding and Subtracting Fractions

    Example 5.1.1.
    Video‚Äâ1‚Äâ
    Adding and subtracting fractions requires a common denominator. The common denominator will be the least common multiple of the original denominators. In general, to add or subtract two fractions:
    1. Rewrite both fractions with a common denominator
    2. Add or subtract the numerators. Keep the denominator the same.
    3. Reduce the fraction to lowest terms.
    1. \(\displaystyle \dfrac{1}{2} + \dfrac{5}{3}\)
    2. \(\displaystyle \dfrac{1}{4}-\dfrac{3}{8}\)
    3. \(\displaystyle 7 + \dfrac{5}{6}\)
    4. \(\displaystyle -1+\dfrac{2}{9}\)
    Solution.
    1. \(\displaystyle \dfrac{1}{2} + \dfrac{5}{3} = \dfrac{3}{6} + \dfrac{10}{6} = \dfrac{13}{6}\)
    2. \(\displaystyle \dfrac{1}{4}-\dfrac{3}{8} =\dfrac{2}{8}-\dfrac{3}{8} = -\dfrac{1}{8}\)
    3. \(\displaystyle 7 + \dfrac{5}{6} = \dfrac{42}{6} + \dfrac{5}{6} = \dfrac{47}{6}\)
    4. \(\displaystyle -1+\dfrac{2}{9} = -\dfrac{9}{9} + \dfrac{2}{9} = -\dfrac{7}{9}\)
    Example 5.1.2.
    Add or subtract the fractions.
    1. \(\displaystyle \dfrac{3}{4}-\dfrac{2}{3}\)
    2. \(\displaystyle \dfrac{2}{5}+\dfrac{7}{10}\)
    3. \(\displaystyle 3 + \dfrac{7}{2}\)
    4. \(\displaystyle \dfrac{1}{3}-1\)
    Solution.
    1. \(\displaystyle \dfrac{3}{4}-\dfrac{2}{3} = \dfrac{1}{12}\)
    2. \(\displaystyle \dfrac{2}{5}+\dfrac{7}{10} = \dfrac{11}{10}\)
    3. \(\displaystyle 3 + \dfrac{7}{2} = \dfrac{13}{2}\)
    4. \(\displaystyle \dfrac{1}{3}-1 = -\dfrac{2}{3}\)

    Subsubsection 5.1.2 Multiplying and Dividing Fractions

    Example 5.1.3.
    Video‚Äâ2‚Äâ
    Multiplying and dividing fractions does not require a common denominator. To multiply two fractions, simply multiply across the numerators and denominators. To divide two fractions, multiply the top or first fraction by the reciprocal of the bottom or second fraction.
    1. \(\displaystyle -\dfrac{1}{2}\cdot \dfrac{15}{19}\)
    2. \(\displaystyle 3\cdot \dfrac{4}{5}\)
    3. \(\dfrac{2/3}{4/5}\) or \(\dfrac{2}{3}\div \dfrac{4}{5}\)
    4. \(\dfrac{3}{9/7}\) or \(3\div \dfrac{9}{7}\)
    Solution.
    1. \(\displaystyle -\dfrac{1}{2}\cdot \dfrac{15}{19} = -\dfrac{15}{38}\)
    2. \(\displaystyle 3\cdot \dfrac{4}{5} = \dfrac{12}{5}\)
    3. \(\dfrac{2/3}{4/5}\) or \(\dfrac{2}{3}\div \dfrac{4}{5}\) equals \(\dfrac{5}{6}\)
    4. \(\dfrac{3}{9/7}\) or \(3\div \dfrac{9}{7}\) equals \(\dfrac{7}{3}\)
    Example 5.1.4.
    Multiply or divide the fractions. Reduce if possible.
    1. \(\displaystyle \dfrac{3}{4}\cdot \dfrac{2}{3}\)
    2. \(\displaystyle \dfrac{2}{5}\div \dfrac{7}{10}\)
    3. \(\displaystyle -5\cdot \dfrac{7}{2}\)
    4. \(\displaystyle \dfrac{1}{10/9}\)
    Solution.
    1. \(\displaystyle \dfrac{3}{4}\cdot \dfrac{2}{3} = \dfrac{1}{2}\)
    2. \(\displaystyle \dfrac{2}{5}\div \dfrac{7}{10} = \dfrac{4}{7}\)
    3. \(\displaystyle -5\cdot \dfrac{7}{2} = -\dfrac{35}{2}\)
    4. \(\displaystyle \dfrac{1}{10/9} = \dfrac{9}{10}\)

    Subsection 5.2 Rules of Exponents

    Subsubsection 5.2.1 Multiplying with Like Bases

    Example 5.2.1.
    Video‚Äâ1‚Äâ
    If we multiply two expressions or numbers that have the same base, we can combine by adding their exponents.
    \begin{equation*} a^m\cdot a^n = a^{m+n} \end{equation*}
    1. \(\displaystyle 5^2\cdot 5^9\)
    2. \(\displaystyle x^{1/2}\cdot x^3\)
    3. \(\displaystyle 2^x\cdot 2^{7x}\)
    Solution.
    1. \(\displaystyle 5^2\cdot 5^9 = 5^{11}\)
    2. \(\displaystyle x^{1/2}\cdot x^3 = x^{1/2 + 3} = x^{7/2}\)
    3. \(\displaystyle 2^x\cdot 2^{7x} = 2^{x+7x} = 2^{8x}\)
    Example 5.2.2.
    Simplify each expression
    1. \(\displaystyle 3^4\cdot 3^{10}\)
    2. \(\displaystyle t^{3/2}\cdot t^4\)
    3. \(\displaystyle e^{x+1}\cdot e^{-2}\)
    Solution.
    1. \(\displaystyle 3^4\cdot 3^{10} = 3^{14}\)
    2. \(\displaystyle t^{3/2}\cdot t^4 = t^{11/2}\)
    3. \(\displaystyle e^{x+1}\cdot e^{-2} = e^{x-1}\)

    Subsubsection 5.2.2 Dividing with Like Bases

    Example 5.2.3.
    Video‚Äâ2‚Äâ
    If we divide two expressions or numbers that have the same base, we can combine by subtracting their exponents:
    \begin{equation*} \dfrac{a^m}{a^n} = a^{m-n} \end{equation*}
    1. \(\displaystyle \dfrac{3^5}{3^2}\)
    2. \(\displaystyle \dfrac{x^7}{x^2}\)
    3. \(\displaystyle \dfrac{x^2}{x^7}\)
    4. \(\displaystyle \dfrac{4^y}{4}\)
    Solution.
    1. \(\displaystyle \dfrac{3^5}{3^2} = 3^{5-2} = 3^3\)
    2. \(\displaystyle \dfrac{x^7}{x^2} = x^{7-2} = x^5\)
    3. \(\displaystyle \dfrac{x^2}{x^7} = x^{2-7}=x^{-5}\)
    4. \(\displaystyle \dfrac{4^y}{4} = 4^{y-1}\)
    Example 5.2.4.
    Simplify each expression
    1. \(\displaystyle \dfrac{2^9}{2^6}\)
    2. \(\displaystyle \dfrac{y^4}{y}\)
    3. \(\displaystyle \dfrac{3^2}{3^3}\)
    4. \(\displaystyle \dfrac{2^x}{2^y}\)
    Solution.
    1. \(\displaystyle \dfrac{2^9}{2^6} = 2^3\)
    2. \(\displaystyle \dfrac{y^4}{y} = y^3\)
    3. \(\displaystyle \dfrac{3^2}{3^3}=3^{-1}\)
    4. \(\displaystyle \dfrac{2^x}{2^y}=2^{x-y}\)

    Subsubsection 5.2.3 Raising a Power to a Power

    Example 5.2.5.
    Video‚Äâ3‚Äâ
    If we raise a power to a power, we can combine by multiplying their exponents.
    \begin{equation*} (a^m)^n = a^{m\cdot n} \end{equation*}
    1. \(\displaystyle (x^4)^5\)
    2. \(\displaystyle (e^t)^3\)
    3. \(\displaystyle (2^3)^2\)
    4. \(\displaystyle (x^{5/3})^{3/5}\)
    Solution.
    1. \(\displaystyle x^{20}\)
    2. \(\displaystyle e^{3t}\)
    3. \(\displaystyle 2^6\)
    4. \(\displaystyle x\)
    Example 5.2.6.
    Simplify each expression
    1. \(\displaystyle (3^2)^5\)
    2. \(\displaystyle (e^x)^2\)
    3. \(\displaystyle (x^9)^{2/9}\)
    Solution.
    1. \(\displaystyle 3^{10}\)
    2. \(\displaystyle e^{2x}\)
    3. \(\displaystyle x^2\)

    Subsubsection 5.2.4 Zero and Negative Powers

    Example 5.2.7.
    Video‚Äâ4‚Äâ
    \begin{align*} a^0=1 \amp a\neq 0 \\ a^{-n}=\dfrac{1}{a^n}\amp \end{align*}
    1. \(\displaystyle 19^0\)
    2. \(\displaystyle 7x^0\)
    3. \(\displaystyle 2^{-3}\)
    4. \(\displaystyle 2x^{-5}\)
    5. \(\displaystyle -\dfrac{6}{t^2}\)
    6. \(\displaystyle \dfrac{3x}{(x+1)^4}\)
    Solution.
    1. \(\displaystyle 19^0=1\)
    2. \(\displaystyle 7x^0=7\cdot 1=7\)
    3. \(\displaystyle 2^{-3}=\dfrac{1}{2^3} = \dfrac{1}{8}\)
    4. \(\displaystyle 2x^{-5}= 2\cdot \dfrac{1}{x^5} = \dfrac{2}{x^5}\)
    5. \(\displaystyle -\dfrac{6}{t^2} = -6t^{-2}\)
    6. \(\displaystyle \dfrac{3x}{(x+1)^4}=3x(x+1)^{-4}\)
    Example 5.2.8.
    Simplify each expression
    1. \(\displaystyle \pi^0\)
    2. \(\displaystyle 3(x^0)\)
    Solution.
    1. \(\displaystyle \pi^0=1\)
    2. \(\displaystyle 3(x^0)=3\)
    Example 5.2.9.
    Rewrite each expression without negative exponents
    1. \(\displaystyle x^{-4}\)
    2. \(\displaystyle (t^2-1)^{-2}\)
    Solution.
    1. \(\displaystyle x^{-4} = \dfrac{1}{x^4}\)
    2. \(\displaystyle (t^2-1)^{-2} = \dfrac{1}{(t^2-1)^2}\)
    Example 5.2.10.
    Rewrite each expression without fractions
    1. \(\displaystyle \dfrac{3}{x^5}\)
    2. \(\displaystyle \dfrac{1}{(x+5)^3}\)
    Solution.
    1. \(\displaystyle \dfrac{3}{x^5} = 3x^{-5}\)
    2. \(\displaystyle \dfrac{1}{(x+5)^3}= (x+5)^{-3}\)

    Subsubsection 5.2.5 Rewriting Radicals

    Example 5.2.11.
    Video‚Äâ5‚Äâ
    \begin{align*} \sqrt[n]{a} \amp a^{1/n}\\ \sqrt[n]{a^m} \amp a^{m/n} \end{align*}
    Rewrite each radical with exponential notation.
    1. \(\displaystyle \sqrt{x}\)
    2. \(\displaystyle \sqrt[3]{x}\)
    3. \(\displaystyle \sqrt[10]{x}\)
    4. \(\displaystyle \dfrac{1}{\sqrt{x}}\)
    5. \(\displaystyle \sqrt{x^5}\)
    6. \(\displaystyle \sqrt[4]{x^7}\)
    7. \(\displaystyle \dfrac{3}{\sqrt[3]{x}}\)
    Solution.
    1. \(\displaystyle \sqrt{x} = x^{1/2}\)
    2. \(\displaystyle \sqrt[3]{x} = x^{1/3}\)
    3. \(\displaystyle \sqrt[10]{x}=x^{1/10}\)
    4. \(\displaystyle \dfrac{1}{\sqrt{x}} = x^{-1/2}\)
    5. \(\displaystyle \sqrt{x^5}=x^{5/2}\)
    6. \(\displaystyle \sqrt[4]{x^7}=x^{7/4}\)
    7. \(\displaystyle \dfrac{3}{\sqrt[3]{x}}=3x^{-1/3}\)
    Example 5.2.12.
    Rewrite each radical with exponential notation
    1. \(\displaystyle 3\sqrt{t}\)
    2. \(\displaystyle \sqrt[5]{x}\)
    3. \(\displaystyle \dfrac{1}{\sqrt[3]{x}}\)
    4. \(\displaystyle \sqrt{x^3}\)
    Solution.
    1. \(\displaystyle 3\sqrt{t} = 3t^{1/2}\)
    2. \(\displaystyle \sqrt[5]{x}=x^{1/5}\)
    3. \(\displaystyle \dfrac{1}{\sqrt[3]{x}}=x^{-1/3}\)
    4. \(\displaystyle \sqrt{x^3}=x^{3/2}\)
